# ✈️ Airline Reservation System

The Airline Reservation System is a GUI-based desktop application developed using Python, Tkinter, and SQLite. The main objective of this project is to provide a simple and efficient platform for users to search flights and book airline tickets digitally. The system focuses on improving user experience through an interactive graphical interface and efficient database management.

The application allows users to create an account by entering personal details such as first name, last name, date of birth, email, and mobile number. After successful registration, users can securely log in using their email and mobile number. The system validates user credentials using an SQLite database to ensure secure access to booking functionalities.

The project includes a flight search module where users can search for flights based on date, source, destination, session, and class. To improve usability, the system provides auto-suggestions while entering search details, helping users quickly find available flights. The search functionality is connected to the flights database table, where matching flights are retrieved and displayed dynamically.

Once a suitable flight is selected, users can confirm ticket booking directly through the application. The booking details are then stored in the database for future reference. The system also validates mobile numbers during registration to ensure correct data entry and prevent invalid inputs.

The Airline Reservation System demonstrates the practical implementation of GUI programming, database connectivity, input validation, and real-time interaction using Python technologies. This project helps in understanding core concepts of desktop application development, database handling, and user authentication systems while providing a real-world airline booking simulation.
